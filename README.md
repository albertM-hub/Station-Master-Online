> **Logbook radioamateur cloud-native moderne** — Application full-stack pour gérer vos QSOs en ligne

## ✨ Features

- ✅ **Authentication** (JWT, registration/login sécurisés)
- ✅ **CRUD QSOs** (Create, Read, Update, Delete avec filtres)
- ✅ **Import ADIF** (Upload fichiers .adi/.adif)
- ✅ **Dashboard** (Stats en temps réel + graphiques)
- ✅ **Responsive** (Mobile-first design)
- ✅ **Dark mode** (Mode sombre natif)
- ✅ **Docker ready** (docker-compose complet)

## 🚀 Quick Start

### Avec Docker (5 min)

\`\`\`bash
git clone https://github.com/albertM-hub/Station-Master-Online.git
cd Station-Master-Online
docker-compose up -d

# Frontend: http://localhost:5173
# API: http://localhost:3000
\`\`\`
